This website is based off an older Curses text based menu. It was made with pure HTML, CSS, and JS.

This website is also a single page application. You may have noticed that when you click a link, the page does not reload. This website uses JQuery to dynamically grab the requested HTML and put it on the page, allowing the website to handle its own loading screens and load all content dynamically.
